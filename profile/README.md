# Starlet Web Engine

**Starlet Web Engine** is a long-term personal exploration of web engine architecture. It prioritizes small, composable libraries with experimentation as a core principle.

> This is not intended to be a publicly released framework, production engine, or drop-in library. It's an experimental lab for personal web development.

## Repositories

### Core
- `starweb-engine` — Canvas, input, game loop, and asset utilities
- `starweb-physics` — SAT and circle collision detection
- `starweb-tween` — Frame-driven tween manager with easing
- `starweb-audio` — Web Audio API playback and management

### Development
- `starweb-sandbox` — Stress test and sandbox environment
